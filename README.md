## おすすめ受講順（このまま上から）
1. **Lesson1**：型と返り値（void/int/float/bool）
2. **Lesson2**：関数（引数/return/切り出し）
3. **Lesson3**：Vector3（座標・速度）
4. **Lesson4**：const参照（const T&）
5. **Lesson5**：コンストラクタ/デストラクタ（RAII入口）
6. **Lesson6**：new/delete（ヒープと寿命）
7. **Lesson7**：unique_ptr（安全な所有）
8. **Lesson8**：循環include（前方宣言の定石）
9. **Lesson9**：Update/Draw とゲームループ
10. **Lesson10**：Singleton
11. **Lesson11**：Factory Method
12. **Lesson12**：Observer
13. **Lesson13**：State（基礎）
14. **Lesson14**：Object Pool
15. **Lesson15**：Flyweight
16. **Lesson16**：State（応用：一時状態→復帰）

1. `project.sln` を開く
2. 起動したいプロジェクト（LessonX）を「スタートアッププロジェクト」に設定
3. 実行（F5 または Ctrl+F5）
